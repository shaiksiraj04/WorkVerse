# 1. About WorkVerse Technologies

WorkVerse Technologies is a virtual software company created to simulate the real software engineering workplace.

Unlike traditional learning platforms, learners join the company as interns and gain practical experience by completing realistic engineering work assignments.

The company exists only inside the WorkVerse platform.

Its purpose is to create an immersive learning experience that closely resembles a modern software organization.

---

# 2. Company Vision

# 9. Work Schedule

The internship is organized into weekly engineering sprints.

Each day includes:

- Morning Briefing
- Work Assignment
- AI Guidance
- Submission
- Performance Feedback

Estimated daily time:

20–30 Minutes

---

# 10. Roles Inside the Company

## CEO

Represents the company vision.

Appears during onboarding and internship completion.

---

## Engineering Manager

Assigns work.

Reviews submissions.

Provides sprint feedback.

Tracks performance.

---

## Senior Backend Engineer

Provides technical guidance.

Answers questions.

Explains concepts.

## Backend Engineering Intern

# 11. Performance Evaluation

Every assignment contributes to:

- XP
- Performance Score
- Skill Progress
- Promotion Progress

Students are evaluated based on:

- Completion
- Code Quality
- Understanding
- Consistency

---

# 12. Promotion System

Promotion is based on:

- Completed Sprints
- XP
- Performance Reviews
- Assignment Completion

Promotions are motivational milestones rather than competitive rankings.

# 13. Communication Style

The company communicates professionally and positively.

"Good morning! Here's today's sprint assignment."

"Excellent work. Let's improve this section together."

"Congratulations on completing another sprint."

Communication should always encourage learning.
